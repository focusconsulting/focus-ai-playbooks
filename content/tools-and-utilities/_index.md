# Tools & Utilities

Quick-reference guides for the AI tools available to the team. Each page covers what the tool is, how to get started, and when to reach for it. If you're new, start with [Claude Team Account]({{< relref "claude-team-account" >}}) to get set up, then [Claude Desktop]({{< relref "claude-desktop" >}}) as your day-to-day workspace.

- [Claude Team Account]({{< relref "claude-team-account" >}}) — Your team's shared Claude account. Log in with SSO and request a seat.
- [Claude Desktop]({{< relref "claude-desktop" >}}) — The main interface for day-to-day AI work: writing, research, brainstorming, and document review.
- [Claude for Chrome]({{< relref "claude-for-chrome" >}}) — A browser extension that lets Claude see and interact with the page you're looking at.
- [Claude Co-working]({{< relref "claude-co-working" >}}) — Hand off well-defined tasks and let Claude work in the background while you focus on other things.
- [Claude Projects]({{< relref "claude-projects" >}}) — Shared workspaces with persistent context for team collaboration.
- [Claude Code]({{< relref "claude-code" >}})
- [Claude + Google Drive]({{< relref "claud-google-drive" >}}) — Connect Claude to your Google Drive so it can read and work with your documents, spreadsheets, and files directly.
