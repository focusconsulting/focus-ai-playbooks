# Ollama

[Ollama](https://ollama.com/) lets you run open-source language models locally on your machine. It handles model downloading, configuration, and serving behind a simple CLI — run `ollama run <model>` and you're in a conversation. It also exposes an OpenAI-compatible API, which means tools like OpenCode can connect to it directly.

For installation and usage, see the [Ollama documentation](https://docs.ollama.com).
