# AnythingLLM

[AnythingLLM](https://anythingllm.com/) is an open-source desktop app that gives you a chat interface on top of any LLM — local or cloud. It supports document embedding via RAG so you can chat against your own files, and includes an agent and workflow builder for automating multi-step tasks. It connects to Ollama, LM Studio, OpenAI, Google Gemini, and many other providers. Available on macOS, Windows, and Linux.

For installation and setup, see the [AnythingLLM documentation](https://docs.anythingllm.com/).
