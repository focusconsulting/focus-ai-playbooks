# AnythingLLM

[AnythingLLM](https://anythingllm.com) is an open-source desktop app that lets you chat with LLMs, attach documents for context, and build agent workflows. It supports multiple LLM providers (cloud and local) and includes built-in RAG for working with your own documents.

## Getting Started

Download AnythingLLM from [anythingllm.com](https://anythingllm.com) and configure it with your preferred LLM provider. See the [AnythingLLM documentation](https://docs.anythingllm.com) for setup instructions.

## When to Use It

AnythingLLM is useful when you want to chat with an LLM against your own documents, or when you want a single interface that can switch between cloud and local model providers. It fills a different niche than coding-focused tools: think document Q&A, research, and knowledge management.
