# Slate

[Slate](https://github.com/NishantJoshi00/slate) is a terminal-based AI coding agent designed for extended, collaborative sessions. It supports tool use, file editing, and multi-step task execution from the command line.

## Getting Started

Install Slate and configure your API key. See the [Slate documentation](https://github.com/NishantJoshi00/slate) for setup instructions.

## When to Use It

Slate is useful when you want an open-source terminal agent that can work through multi-step coding tasks with tool access, similar to Claude Code but with different trade-offs.
