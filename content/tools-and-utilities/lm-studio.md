# LM Studio

[LM Studio](https://lmstudio.ai/) is a desktop app for running language models locally with a GUI. It lets you browse, download, and run models without touching the command line, and exposes a local API server that other tools can connect to. Available on macOS, Windows, and Linux.

For setup and developer usage, see the [LM Studio documentation](https://lmstudio.ai/docs/developer).
